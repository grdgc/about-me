### Setup

```sh
$ git clone https://github.com/sethbergman/about-me.git && cd about-me
```

### Install dependencies

```sh
$ npm i # or use yarn
$ yarn
```

### Start the app

```sh
$ npm start
```

##### Visit [http://0.0.0.0:5000](http://0.0.0.0:5000) in your browser.
